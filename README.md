#VARIAVEIS
Tipos que utilizei:

String (Texto): Para nomes, mensagens e CEPs. É tudo que vai entre "aspas".

Int (Inteiro): Para números sem virgula, tipo contagem de pacotes ou idade.

Float (Real): Para quando a precisão importa, tipo notas ou quilos.

Boolean (Lógico): O famoso "chaveamento" (True/False). É o que define as decisões do sistema.
